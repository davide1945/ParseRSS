#### Usage

#### Gradle Dependency

Add jitpack repository in your **root** `build.gradle` at the end of repositories:
```gradle
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
And then add the dependency
```gradle
implementation "com.github.muhrifqii:ParseRSS:$version"
```
